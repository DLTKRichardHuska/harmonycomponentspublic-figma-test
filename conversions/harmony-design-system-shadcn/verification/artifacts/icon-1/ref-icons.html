Icon Selection Guide

Icon Hierarchy for Application Development

When building applications with this design system, follow this priority order:

  1. Check Heroicons First - 284 outline icons for common UI needs
    <Icon name="home" />
  2. Use Tabler Icons as Fallback - 4,600+ icons when Heroicons doesn't have what you need
    <i class="ti ti-brand-github"></i>
    Browse Tabler Icons →
  3. Custom Icons for Domain-Specific Needs - Project management, risk management, etc.
    <Icon name="gantt-chart" />

Note: This design system documentation uses only Heroicons and Custom icons for consistency. Tabler is available as a developer tool when needed.

Accessibility

Icon Accessibility

All icons (Heroicons, Tabler, and Custom) include aria-hidden="true" by default. When using icons without text labels, add an aria-label to the parent button or link for screen reader users.

<button aria-label="Edit document">
  <Icon name="pencil" />
</button>

<button aria-label="Delete item">
  <Icon name="trash" />
</button>
