Accessibility

ARIA Attributes

Spinners use role="status" and aria-label (default "Loading") so assistive tech announces the loading state. Override aria-label for more specific context (e.g. "Loading content").

Screen Reader Support

A visually hidden label mirrors aria-label. Prefer descriptive labels when the surrounding UI does not already explain what is loading.

Animation Preferences

Spin animation respects prefers-reduced-motion via motion-reduce:animate-none.
