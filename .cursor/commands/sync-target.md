# /sync-target

Advanced: one-shot apply to a target conversion playbook. **Prefer `/conversion-agent`** for status, plan, and approval.

Cursor-only — not npm build or CI.

## Input

**Target id** (required). Optional scope (e.g. `foundation`, `Button`).

## Instructions

1. Load **harmony-conversion** and **sync-target** skills.
2. Read target `conversion/SKILL.md` and follow apply steps for scope.
3. For plan/status/full workflow, use **conversion-agent** instead.
